My React Portfolio
Welcome to my personal portfolio website! This project showcases my skills, experience, and projects in web development. Built with React.js, this portfolio highlights my journey as a Java Full-Stack Developer.

Features
Responsive Design: The portfolio is fully responsive, ensuring a great user experience on both desktop and mobile devices.
Modern UI/UX: Sleek and modern design with smooth animations and intuitive navigation.
Interactive Elements: Includes interactive components like an animated cursor and scroll-to-top functionality.
Projects Showcase: Detailed view of my projects with descriptions, technologies used, and live demo links.
Blog Section: A dedicated section for my blog posts where I share insights and experiences.
Contact Form: A simple and effective way for visitors to get in touch with me.
Technologies Used
Frontend: React.js, React Router, Styled Components, Framer Motion
Styling: CSS, Bootstrap
Hosting: Firebase Hosting
Utilities: EmailJS for contact form integration, GitHub Actions for CI/CD
